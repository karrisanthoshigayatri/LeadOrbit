# 🚀 MVP Technical Design (Lean but Scalable)

## Backend
- Django
- Django REST Framework
- PostgreSQL
- Celery
- Redis
- Firebase Admin SDK

## Frontend
- Stitch MCP Templates
- Tailwind CSS
- Alpine.js (light interactivity)

## Messaging Integrations
- Email via SMTP relay
- SMS provider API
- WhatsApp Cloud API
- LinkedIn automation microservice

## AI Layer
- Google Gemini API (Free Tier)
- Structured prompt templates
- Fallback deterministic templates

---

## MVP Scope

1. Organization-based login
2. Lead CSV upload (async)
3. Campaign creation
4. Multi-step Email sequence
5. AI-generated personalization
6. Stop-on-reply
7. Basic analytics dashboard
8. Unsubscribe handling
